Tutoriel sur les blocs - Comment modifier les blocs avec la boîte à outils et l'espace de travail
--------------------------------------------------------------------------------------------------

Si vous souhaitez ajouter ou supprimer des blocs de votre tâche, vous pouvez le faire en utilisant soit l'**interface graphique**, soit les **fichiers XML**. Les blocs spécifiques à un labyrinthe se trouvent dans la catégorie Bibliothèque de blocs, et sont définis dans le fichier **"blocks.js "** (sous YourTask/public/), qui ne doit pas être modifié, sauf si vous souhaitez y ajouter de nouveaux blocs personnalisés.

Pour créer un nouveau bloc personnalisé, veuillez vous référer à ce tutoriel : TODO mettre un lien vers "blocs avancé"
